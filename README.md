## Getting Started

### Prerequisites
- Java installed (JDK 8+ recommended)
- Any IDE (IntelliJ IDEA / Eclipse / VS Code) or terminal

### Clone the repository
```bash
git clone https://github.com/Rohittt-Malviya/DSA-WITH-JAVA.git
cd DSA-WITH-JAVA
```

### Run a Java file
If your files are plain `.java` files (not using Maven/Gradle), you can run like:

```bash
javac FileName.java
java FileName
```

> If your code is inside packages (e.g., `package dsa.arrays;`), compile/run from the project root with the correct classpath/package structure.
